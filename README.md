# Ecommerce-Sales-Analysis

E-commerce Sales Analysis using Power BI

Built a comprehensive Power BI dashboard using an e-commerce dataset containing nearly 10,000 orders. The dashboard provides insights into sales performance, profitability, order trends, regional distribution, and product category performance.

**Key Features:**

Sales, Profit, Orders, Quantity, and Profit Margin KPIs
Year-over-Year (YoY) performance analysis
Interactive Map Chart for regional sales insights
Category and Sub-Category performance analysis
Top 5 Sub-Categories visualization
Combo, Pie, Waterfall, and KPI charts
Dynamic filtering and business trend analysis

**Tools Used:**
Power BI, Excel, DAX, Data Modeling, Power Query
